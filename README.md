# Overview

Simple utility for downloading a selected list of plugins for your JetBrains products.

# Usage

Install using:

```shell
git clone git@github.com:doronz88/download-jetbrains-plugins.git
cd download-jetbrains-plugins
python3 -m pip install -r requirements.txt
```

Now download using:

```shell
python3 download.py /path/to/output/dir
```
